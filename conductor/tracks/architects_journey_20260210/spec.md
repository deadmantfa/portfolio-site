# Specification: Architect's Journey MVP

## Overview
This track focuses on creating the core 'wow factor' of the portfolio: an immersive 3D landing page that seamlessly transitions from a visionary hero section into a career timeline, all while maintaining top-tier SEO and performance.

## Key Features
- **3D Hero Section:** A React Three Fiber (R3F) scene featuring a 'Visionary Architect' aesthetic (e.g., dynamic grid or starfield).
- **Interactive Timeline:** A scroll-triggered 3D transition that visualizes 20 years of experience.
- **Global Navigation:** Persistent UI for 'About', 'Work', 'Contact', and a 'Download CV' CTA.
- **SEO Foundations:** Next.js Metadata API integration and semantic HTML structures.

## Technical Constraints
- Next.js 15+ (App Router)
- React Three Fiber & Drei for 3D
- Tailwind CSS for layout
- Framer Motion for UI transitions
